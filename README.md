## 🗺️ 7-Point Learning Path

1. **Python + Java** — OOP, async, collections, exceptions (Month 1)
2. **Kaggle ML/DL/NLP/CV** — All 4 courses with exercises (Months 2 & 4)
3. **Striver A2Z Sheet** — Java, 3 problems/day throughout all 6 months
4. **LangChain + PostgreSQL** — RAG agent, pgvector, LCEL (Month 2)
5. **System Design basics** — CAP, scaling, rate limiting, Redis (Months 2 & 4)
6. **FastAPI + RAG course** — Endpoints, JWT, async DB, full RAG pipeline (Month 2)
7. **OS + DBMS + CN basics** — Processes, ACID, TCP/IP, HTTP/2 (Month 4)

1. Fork this repo
2. Go to [app.netlify.com](https://app.netlify.com) → **Add new site → Import from Git**
3. Select this repo, branch `main`, publish dir `.`
4. Deploy — works across all devices instantly

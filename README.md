# 🤖 AI Engineer Roadmap — PWA

A free, fully offline-capable **Progressive Web App** for anyone who wants to become an Applied AI Engineer. No sign-up, no backend, no cost. Just open it and start.

> Built for learning Python → ML → LLMs → RAG → Agents and tracking DSA progress — all in one place.

---

## 📱 Pages

| Page | What it does |
|------|--------------|
| [roadmap.html](roadmap.html) | 8-phase AI Engineer roadmap with topics, resources, and projects for each phase |
| [dsa.html](dsa.html) | **Striver A2Z DSA Sheet** — 222 problems, 15 chapters, LeetCode & GFG links, ⭐ star to bookmark, progress tracker |
| [resources.html](resources.html) | Best free & paid resources per skill — curated, no overwhelm |
| [timetable.html](timetable.html) | Daily study schedule (Weekday / Weekend) |
| [review.html](review.html) | Monthly milestone checklists + reflection journal |

---

## ⚡ Features

- **Works offline** — Pure HTML + CSS + Vanilla JS. Zero frameworks, zero build step.
- **Installable PWA** — Add to home screen on any phone. Works like a native app.
- **Progress saved locally** — All checkboxes, stars, and journal entries save to `localStorage` in your browser.
- **Light & Dark theme** — Toggle in the top nav. Preference saved automatically.
- **DSA Sheet** — Full Striver A2Z tracking with problem number, LeetCode link, GFG link, difficulty badge, ⭐ bookmark, and done checkbox.
- **Monthly Review** — Honest end-of-month checklists per phase + journal for reflection.

## 🚀 Use It (No Setup Needed)

### Option 1 — Open directly in browser
```
git clone https://github.com/hemasaivattikuti25/roadmap.git
cd roadmap
open roadmap.html
```

### Option 2 — Deploy free on Netlify
1. Fork this repo
2. Go to [app.netlify.com](https://app.netlify.com) → **Add new site → Import from Git**
3. Select your fork, branch `main`, publish directory `.`
4. Deploy — live in under 60 seconds

### Option 3 — Install as PWA
Open the deployed URL in Chrome/Safari → **Add to Home Screen** → works offline like an app

---

## 📦 Data & Privacy

- All progress (DSA ticks, stars, journal entries) stores in **your browser's localStorage only**
- Nothing is sent to any server — there is no server
- Clearing browser data will reset progress
- Progress is **device-specific** — does not sync across devices
